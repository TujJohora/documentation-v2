Homepage Setup

Go to Admin -> Appearance -> Theme options (tab Page) to select a homepage.

Customize homepage

Go to Admin -> Pages -> Select Homepage

Home Slider

It's shortcode simple-slider in the page content.

You can change slider or set ads / display newsletter form if you want.

In this case, it's using the slider with key home-slider-1. You can find it in Admin -> Simple Sliders.

You can modify Home Slider 1 or change the key of shortcode to use other sliders.

Menu

Go to Admin -> Appearance -> Menus (select Main menu)

Drag and drop to change menu items.

Header menu

Go to Admin -> Appearance -> Menus (select Header menu)

Footer menus

Go to Admin -> Appearance -> Menus (modify menu Company, Information & Product categories)

You can change or remove those menus in Admin -> Appearance -> Widgets.

Theme Options

Go to Admin -> Appearance -> Theme options.

Click to your language on the right side if you want to setup theme options in other language.

Widgets

Site features

You can modify it in Admin -> Appearance -> Widgets.

Footer widgets

Go to Admin -> Appearance -> Widgets.

You can click on "Delete" button to remove widget from footer sidebar or change the name, selected menu.
If you want to add more widgets, just drag and drop from the left side to sidebar.

Product sidebar widgets

Go to Admin -> Appearance -> Widgets.

If you want to disable widgets in the product detail page, you can set product layout in Admin -> Appearance -> Theme options (tab Ecommerce) and change Product Single Layout to full width.

Blog sidebar widgets

Go to Admin -> Appearance -> Widgets.

If you want to disable widgets in the blog page, you can set blog layout in Admin -> Appearance -> Theme options (tab Blog) and change Blog Single Layout to full width.

And go to Admin -> Pages (select page Blog) then set Template to full width.

In admin panel, go to Appearance -> Custom JS.

You can add your CSS/JS directly in platform/themes/[your-theme]/partials/footer.blade.php.

Multi-language

Setting Up Languages

Navigate to admin panel and click to Settings -> Languages.

Changing the Default Language

Adding a New Language
To change the default language, you first need to add the desired language. In this example, we will add **Tiếng Việt ** (Vietnamese) language.

Setting the Default Language
Once the language is added, click on the Is default? button in the Tiếng Việt row to make it the default language.

Disabling Multi-language

Navigate to Plugins -> Installed Plugins and type language in the search bar. Find the Language and Language Advanced plugins and disable them to disable multi-language functionality.

Troubleshooting

Service Not Enabled Error
If you encounter the "Service is not enabled" error:

1. Enable Google Analytics Data API

Go to your Google Cloud Console, select your project, search for "APIs & Services", and enable the "Google Analytics Data API".

2. Setup Timezone and Clear Cache

In admin panel, go to -> Settings -> General and set your local timezone. Then, go to Platform Administration -> Cache Management and clear your site cache.

Currencies

Currency Settings

To change currency settings go to Settings -> Ecommerce -> Currencies.

Currency Exchange Rates

Automatically Updating Exchange Rates

Without the Use exchange rate from API feature, the customer needs to do their own research on the exchange rate and manually update the exchange rate in the currency settings.
We provide two options for automatically updating the exchange rate:

  • API Layer
  • Open Exchange Rates

In the API provider field, select the API provider you want to use and enter their API key.

Manually Updating Exchange Rates

If you prefer to manually update the exchange rate, select None in the API provider field and disable the Use exchange rate from API feature.

Newsletter

Connect Mailchimp

Configure and manage newsletter functionality.

  • Go to Mailchimp and create an account.
  • Create a new list in Mailchimp https://us20.admin.mailchimp.com/lists.
  • Go to Account -> Extras -> API keys
  • Copy API key and paste to Admin -> Settings -> Newsletter -> Mailchimp API key.
  • Copy List ID and paste to Admin -> Settings -> Newsletter -> Mailchimp List ID, then click Save. Or just click Save setting, and it will auto get List ID.
